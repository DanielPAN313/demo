# modules

Product feature modules that combine UI, state, and domain logic.

Recommended initial folders/files:

- `rhythm/`: today rhythm timeline and rhythm status.
- `reschedule/`: state-aware rescheduling flow.
- `firewall/`: attention firewall and notification grouping.
- `briefing/`: morning briefing and night digest.
- `agentChat/`: user command input and agent replies.

This project currently keeps feature UI under `features/`. New modules can either be added here directly or migrated from `features/` as they become larger.
